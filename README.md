## How to Use

1. **Share a Story**: Fill in the title and content, then click "Submit Story"
2. **Rate Stories**: Enter a rating (1-10) and click "Rate" for any story
3. **View Ratings**: See the average rating and number of ratings for each story

## Data Privacy

- Stories are stored **only in your browser's localStorage**
- Each browser/device has its own separate data
- Clearing browser data will delete all stories
- No data is sent to any server
